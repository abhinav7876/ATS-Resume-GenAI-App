A Applicant Tracking System (ATS) score checker that evaluates resumes against a given job description. The project parses resumes, extracts key skills, compares them with JD keywords, and generates an ATS-style match score. Ideal for understanding NLP, text processing, and building job-matching tools.

# How to run?
### STEPS:

Clone the repository

```
git clone https://github.com/abhinav7876/ATS-Resume-GenAI-App.git
```
### STEP 01- Create a conda environment after opening the repository

```
conda create -p venv python==3.10 -y
```

```
conda activate venv/
```


### STEP 02- install the requirements
```
pip install -r requirements.txt
```
# Finally run the following command
python app.py
